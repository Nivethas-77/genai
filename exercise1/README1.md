Exercise 1: Basic LLM Call using Groq API

Aim

To perform a basic Large Language Model (LLM) call using Groq API.

Overview

This exercise demonstrates how to connect to a Generative AI model using the Groq API and generate a response for a given prompt.

Technologies Used

* Python
* Groq API
* OpenAI-compatible client
* python-dotenv

Description

The program loads the API key from a `.env` file, connects to the Groq model, sends a prompt related to Generative AI, and displays the response in the terminal.

Input Prompt

Explain Generative AI, its types, and real-world applications.
